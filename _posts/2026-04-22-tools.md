## なにはなくとも画像生成

[**ImageFx**](https://labs.google/fx/ja/tools/image-fx) ・・ これまでは大変お世話になってました。  
結構適当なプロンプトでも精度の高いimageを生成してくれて助かってました。  
26年4月でサービス完了と発表されて残念ですが今後は同じgoogle系のflowのお世話になります。  

[**chatGPT**](https://chatgpt.com/) ・・ 対話型でいろんなことをしてくれるgeminiと双璧のAIサービス。  
弊社ではここの有料プランからsoraで動画が生成できるので利用してました。  
SORA2もサービス終了がアナウンスされているのでどうなるのやら・・様子を見て次のツールを決めます。

**PowerPoint** ・・ 誰もが知ってるパワポは主にチラシ作りのツールになってます。  

**Clipchamp** ・・ Windows11付属の動画編集ツール、良いツールだったんですが最近の更新でちょっとね・・  

**AviUtl2** ・・ フリーの動画編集ツール、フリーのものはたくさんありますがUI古くても使いやすいですね。

## こんなツールも使いますよ！　programming language

**processing** ・・ java製のグラフィックス生成ツール  
java言語で様々な画像を作成できる面白いツールです。  
[このインスタグラムのリール](https://www.instagram.com/p/DUz0QKegYu8/)の最初の抽象的な図形の画像生成にも使ってますよ！  
上記の画像のプログラムはこんなふうです。35行ほどのプログラムで動画が作れるって面白いよね。

```java
float x = 0.1, y = 0.1;

//static final float a =  0.45, b = 1.9;
//static final float a =  1.0, b = 0.85;
//static final float a =  1.0, b = 0.9;
static final float a =  1.25, b = 0.75;

void setup() {
  size(800, 800);
  blendMode(ADD);
  background(0);
  stroke(124, 155, 255, 50);
}

void draw() {
  float _x, _y;

  for (int i = 0; i < 1000; i++) {

    _x = (1 + a * b) * x - b * x * y;
    _y = (1 - b) * y + b * x * x;

    point(_x * 100 + width*0.5, - _y * 100 + height*0.75);

    x = _x;
    y = _y;
  }

   saveFrame("frames/####.tif");
  if (frameCount >= 1800) { 
    exit();
  }
}
```
