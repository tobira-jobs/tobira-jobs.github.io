## こんなツールも使いますよ！ 【コンピュータ言語たち】

**access** ・・ microsoft officeに入ってたりなかったりちょっと親しみのないツールって印象

- でもこの子、使いようによっては強力なツールにもなるしただただ時間と手間を食うものにもなる癖のあるツールですね。
- excelとの違いが分からない人もいるかもしれませんがexcelは極論で言えば表の形式を母体としたツール  
  accessは表の形式のデータを組み合わせたり加工したりしながら一つのデータを何度も使いまわすためのツールです。
  
- この機会にaccessをもう一度学んでおきたい人や復習したい人のために有効そうなサイトを２つ紹介します。
1. [もう一度学ぶMS-Access](https://www.accessdbstudy.net/) ・・ accessの基本操作から各パーツの仕組みの説明も丁寧でわかりやすいです。
1. [Access学習ロードマップ](https://road-to-access.com/access_roadmap/) ・・ 上記のものよりはプログラミング環境としてVBAの説明にも力がはいっている印象です。

機会があればvbaについてはまた場所を改めて触れたいと思いますが既知の情報が多い中、何に焦点を当てるか考えてページを作成します。

**processing** ・・ java製のグラフィックス生成ツール  
java言語で様々な画像を作成できる面白いツールです。  
ちょっと凝った動きをさせるのにOOPの練習にもなってプログラム初級者にもおすすめ  
[このインスタグラムのリール](https://www.instagram.com/p/DUz0QKegYu8/)の最初の抽象的な図形の画像生成にも使ってますよ！  
上記の画像のプログラムはこんなふうです。35行ほどのプログラムで動画が作れるって面白いよね。

```java
float x = 0.1, y = 0.1;

//static final float a =  0.45, b = 1.9;
//static final float a =  1.0, b = 0.85;
//static final float a =  1.0, b = 0.9;
static final float a =  1.25, b = 0.75;

void setup() {
  size(800, 800);
  blendMode(ADD);
  background(0);
  stroke(124, 155, 255, 50);
}

void draw() {
  float _x, _y;

  for (int i = 0; i < 1000; i++) {

    _x = (1 + a * b) * x - b * x * y;
    _y = (1 - b) * y + b * x * x;

    point(_x * 100 + width*0.5, - _y * 100 + height*0.75);

    x = _x;
    y = _y;
  }

   saveFrame("frames/####.tif");
  if (frameCount >= 1800) { 
    exit();
  }
}
```

[こちらのインスタのリールの最初の四角形が次々に現れる動画](https://www.instagram.com/p/DUCxZAugQF7/)　も同じく processingで作りました。

programのソースは以下
```java
int i;
int j;
float l;
color[] palette = {
  color(0, 0, 0),// 黒
  color(0, 0, 255),// 青
  color(255, 0, 0),// 赤
  color(255, 0, 255),// マゼンタ
  color(0, 255, 0),// 緑
  color(0, 255, 255),// 水
  color(255, 255, 0),// 黄
  color(255, 255, 255),// 白
};

void setup() {
  j=0;
  i=0;
  size(450, 800);
  background(0);
}
void draw() {
    //stroke(0);
    //fill(palette[int(random(0,8))]);
    stroke(palette[int(random(0,8))]);
    strokeWeight(4);    
    fill(220);
    l=random(10,100);
    rect(random(-50, 540), random(-50, 960), l, l , 10);
  
   saveFrame("frames/####.png");
 if (frameCount >= 1000) {
   exit();
 }
}
```

他にも言語を使ってグラフィックの動画を生成して広告の中に入れようと試みてますが楽しいですよ。
